# 安装
安装前要先安装依赖

- [https://github.com/baresip/re](https://github.com/baresip/re)
- [https://github.com/baresip/rem](https://github.com/baresip/rem)
- openssl
```javascript
git clone https://github.com/baresip/baresip
cd baresip
make
sudo make install
```
