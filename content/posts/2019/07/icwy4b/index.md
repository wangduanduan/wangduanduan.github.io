# 梦与诗
**胡适**<br /> <br />醉过才知酒浓<br />爱过才知情重<br />你不能做我的诗<br />正如我不能做你的梦

# 情歌
**刘半农**<br />天上飘着些微云<br />地上吹着些微风<br />啊！<br />微风吹动了我的头发<br />教我如何不想她？

# 沙扬娜拉 赠日本女郎
**徐志摩**<br />最是那一低头的温柔<br />像一朵水莲花不胜凉风的娇羞<br />道一声珍重<br />道一声珍重<br />那一声珍重里有蜜甜的忧愁<br />沙扬娜拉！

# 再别康桥
