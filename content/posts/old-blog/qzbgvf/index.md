# 1. 实验准备
- T450笔记本

# 2. 进入BIOS

1. 重启电脑
2. 一直不停按enter
3. 按F1
4. 选择Keyboard/mouse

# 3. 恢复F1-F2恢复原始功能:
fn and ctrl key swap **[enabled]**

# 4. 切换ctrl和ctrl的位置:
F1-F12 as primary function **[enabled]**
